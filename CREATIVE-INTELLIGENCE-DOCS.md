## 🧠 Sistema de 3 Fases

### FASE 1: ANÁLISE CRIATIVA PROFUNDA
**Arquivo**: `creative-intelligence.ts` → `analyzeCreativeDirection()`

O sistema analisa o prompt e determina:

#### 1. **Nicho Detectado**
- Dental, Luxury, Construction, Tourism, Law, Fashion, Restaurant, SaaS, General

#### 2. **Direção Emocional**
Exemplos:
- Dental: `trust + serenity + transformation`
- Luxury: `exclusivity + sophistication + desire`
- Tourism: `wanderlust + experience + emotion`
- Law: `authority + confidence + trust`
- Fashion: `bold + artistic + experimental`

#### 3. **Posicionamento de Marca**
- Ultra-premium
- Premium healthcare
- Corporate-premium
- Experiential
- Professional excellence
- Avant-garde
- Culinary excellence
- Cutting-edge technology

#### 4. **Linguagem Visual**
Exemplos:
- Dental: `clean + calm + elegant + medical precision`
- Luxury: `minimal + elegant + timeless + editorial`
- Tourism: `immersive + cinematic + editorial + emotional`
- Fashion: `editorial + asymmetric + bold + artistic`

#### 5. **Audiência Alvo**
- Health-conscious individuals seeking confidence
- Affluent individuals seeking exceptional quality
- Business decision-makers seeking reliability
- Experience-seekers wanting memorable journeys
- Style-conscious individuals seeking uniqueness

#### 6. **Estratégias Derivadas**
- **Hierarchy Style**: Como organizar informação
- **Layout Strategy**: Abordagem de composição
- **Typography Direction**: Fontes + pesos + tracking
- **Spacing Rhythm**: Sistema de espaçamento
- **Visual Storytelling**: Narrativa visual
- **Color Psychology**: Paleta emocional
- **Interaction Style**: Micro-interações

---

### FASE 2: COMPOSIÇÃO VISUAL
**Arquivo**: `creative-intelligence.ts` → `composeVisualStructure()`

Baseado na análise, o sistema decide:

#### 1. **Estilo de Hero**
- `cinematic-fullscreen`: Full-screen imersivo
- `minimal-centered`: Minimalista centrado
- `editorial-split`: Split editorial
- `immersive-layered`: Camadas imersivas
- `bold-asymmetric`: Assimétrico ousado

#### 2. **Ritmo de Secções (ÚNICO)**
Cada geração tem ritmo diferente. Exemplos:

**Luxury - Variação 1:**
```
1. hero-minimal
2. statement-oversized
3. product-editorial
4. craftsmanship-story
5. cta-elegant
```

**Luxury - Variação 2:**
```
1. hero-editorial
2. philosophy-minimal
3. collection-asymmetric
4. heritage-story
5. contact-refined
```

**Dental - Variação 1:**
```
1. hero-trust
2. treatments-grid
3. technology-showcase
4. team-professional
5. testimonials-proof
6. cta-appointment
```

**Dental - Variação 2:**
```
1. hero-transformation
2. about-story
3. services-cards
4. results-gallery
5. faq-trust
6. contact-easy
```

#### 3. **Pacing Visual**
- `slow-immersive`: Cinematográfico, imersivo
- `fast-dynamic`: Dinâmico, energético
- `slow-calm`: Calmo, sereno
- `balanced`: Equilibrado

#### 4. **Nível de Assimetria**
- `high`: Editorial, artístico
- `moderate`: Moderno, balanceado
- `low`: Estruturado, corporativo

#### 5. **Elementos Cinematográficos**
Baseado na emoção:
- Ken Burns zoom
- Parallax layers
- Scroll-triggered reveals
- Fade-in sequences
- Broken grid overlaps
- Asymmetric positioning
- Bold color blocks
- Experimental layouts
- Subtle fades
- Elegant transitions
- Minimal motion
- Refined reveals

**Críticos** (Falha automática):
- DOCTYPE correto
- Tailwind CDN
- Google Fonts
- ZERO Lorem Ipsum (-25 pontos)
- Meta viewport
- Title + description

**Design Moderno**:
- CSS variables
- Animações CSS
- IntersectionObserver
- Backdrop-blur
- Gradientes
- Hover effects (min 5)
- Transitions

**Estrutura**:
- Secções (min 4)
- Navbar
- Footer
- Mobile menu

**Responsividade**:
- Breakpoints (min 10)

**Qualidade de Código**:
- Tags semânticas
- Espaçamento (py-20+)
- JavaScript funcional
- Comprimento (>5000 chars)
- Imagens (min 3)
- CTAs (min 2)

**Padrões Modernos**:
- Rounded corners (2xl+)
- Shadows (lg+)
- Grid assimétrico
