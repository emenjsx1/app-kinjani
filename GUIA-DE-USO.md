## 🚨 Troubleshooting

### Problema: "Site parece genérico"
**Solução**: Adicione mais contexto no prompt
```
Antes: "Website para clínica"
Depois: "Landing page para clínica dentária premium em Lisboa, 
         focada em implantes, público 30-50 anos"
```

### Problema: "Cores não combinam com marca"
**Solução**: Especifique no prompt
```
"Landing page para clínica dentária, 
 paleta de cores: azul marinho e dourado"
```

### Problema: "Falta uma secção específica"
**Solução**: Mencione no prompt
```
"Website para restaurante com secções: 
 menu, chef, galeria, reservas, localização"
```
